# Xaymaca-IO / .github

This is the organization-level configuration repository for **Xaymaca-IO** on GitHub.

The interesting file is [`profile/README.md`](./profile/README.md) — GitHub renders it on the organization's public profile at [github.com/Xaymaca-IO](https://github.com/Xaymaca-IO), visible above the list of pinned repositories.

This repository may, over time, also house organization-wide GitHub configuration: default issue templates, pull request templates, contribution guidelines, code-of-conduct files, and shared CI workflow templates. For now it exists solely to render the org profile.

## Structure

```
.
├── profile/
│   └── README.md      ← the org profile (rendered at github.com/Xaymaca-IO)
└── README.md          ← this file (rendered when someone opens the repo directly)
```

## To preview the profile README

Render `profile/README.md` in any Markdown viewer, or push to GitHub and visit the org page.
